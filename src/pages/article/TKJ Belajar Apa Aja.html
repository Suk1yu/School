Sebuah catatan perjalanan dari Kabel Fisik hingga Baris Kode.

Banyak yang mengira anak Teknik Komputer dan Jaringan (TKJ) hanya bisa memperbaiki komputer rusak. Padahal, realitanya jauh lebih kompleks. Selama pembelajaran produktif, kami telah berevolusi dari sekadar "tukang pasang kabel" menjadi "arsitek jaringan" yang juga memahami bahasa pemrograman.

Berikut adalah deskripsi mendalam tentang 4 pilar utama yang telah kami pelajari dan kuasai sejauh ini:

1. Fondasi Infrastruktur: Crimping & Instalasi LAN

Pelajaran pertama dan paling fundamental adalah membangun jalan raya data itu sendiri. Sebelum internet bisa mengalir, media fisiknya harus siap. Di sini kami belajar bukan sekadar memotong kabel, tapi memahami standar industri.

Apa yang dikuasai?

  • Presisi Warna: Menghafal dan mempraktikkan urutan warna kabel standar TIA/EIA-568A dan T568B (Putih Orange, Orange, Putih Hijau, Biru...).
  • Teknik Crimping: Menggunakan tang crimping untuk menanamkan kabel UTP ke konektor RJ-45. Tantangannya adalah memastikan 8 pin tembaga menyentuh inti kabel dengan sempurna.
  • Cable Management: Memasang kabel agar rapi di dinding atau tray, serta melakukan pengujian konektivitas menggunakan LAN Tester.

2. Otak Jaringan: Setting Mikrotik Wireless

Setelah kabel terpasang, kami beralih ke perangkat yang mengatur lalu lintas data: Router Mikrotik. Fokus utamanya adalah bagaimana menyebarkan internet tanpa kabel (Nirkabel/Wireless) dengan jangkauan dan keamanan yang optimal.

Kami belajar mengonfigurasi Mikrotik sebagai:

  • Access Point (AP): Pusat pemancar sinyal Wi-Fi untuk klien (HP/Laptop).
  • Station/Client: Penerima sinyal Wi-Fi (menangkap internet dari sumber lain).
  • Point to Point (PTP): Teknik "menembak" sinyal Wi-Fi jarak jauh antar gedung.

Kami juga belajar mengamankan jaringan menggunakan Security Profile (WPA/WPA2) agar Wi-Fi tidak mudah dibobol orang asing.

3. Manajemen User: Hotspot System

Ini adalah materi yang paling terasa manfaatnya di dunia nyata (seperti usaha RT/RW Net atau Kafe). Kami belajar membuat sistem di mana pengguna tidak bisa langsung internetan hanya dengan tahu password Wi-Fi, tapi harus Login terlebih dahulu.

Fitur Hotspot yang dipelajari:

  • Manajemen User Profile: Membedakan kecepatan internet (bandwidth) antara Guru dan Siswa.
  • Voucher System: Membuat akun yang memiliki masa aktif (misal: voucher 3 jam atau 1 hari).
  • Walled Garden: Mengizinkan akses ke situs tertentu (misal: web sekolah) meskipun user belum login.

4. Sentuhan Modern: Coding HTML & Python

TKJ zaman now tidak bisa lepas dari koding. Kami mempelajari dua bahasa pemrograman penting untuk menunjang pekerjaan jaringan:

a. HTML Dasar (Untuk Tampilan)

Kami menggunakan HTML bukan untuk menjadi Web Developer murni, tapi untuk mempercantik tampilan Login Page Hotspot Mikrotik. Daripada tampilan login standar yang membosankan, kami bisa mendesain sendiri halaman login dengan logo sekolah atau instruksi yang jelas.

<h1>Selamat Datang di Wi-Fi Sekolah</h1> <form action="$(link-login-only)" method="post"> <input type="text" name="username" placeholder="Kode Voucher"> <button>Login</button> </form>

b. Python (Untuk Otomasi)

Python adalah bahasa masa depan bagi Network Engineer. Kami belajar logika dasar Python untuk kebutuhan otomasi jaringan (Network Automation). Tujuannya agar kelak kami bisa mengonfigurasi 100 router sekaligus hanya dengan satu skrip, tanpa harus menyetingnya satu per satu secara manual.

# Contoh logika sederhana Python di jaringan ip_address = "192.168.1.1" status = "Online" if status == "Online": print(f"Router {ip_address} siap digunakan") else: print("Cek kabel LAN!")

Kesimpulan

Dari memotong kabel fisik, menyeting gelombang radio (wireless), mengatur hak akses pengguna (hotspot), hingga menulis kode program (HTML/Python). Paket keahlian yang kami pelajari membentuk kami menjadi teknisi yang adaptif: kuat di hardware, cerdas di konfigurasi, dan kreatif di software.
