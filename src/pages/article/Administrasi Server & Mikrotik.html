‚ÄãAdministrasi Server & Mikrotik: Mengelola "Otak" Jaringan

5 min Read November 4, 2025 By Admin

Jika kabel dan fiber optik adalah "tubuh" dari sebuah jaringan, maka Sistem Administrasi Server dan Mikrotik adalah "otaknya". Di jurusan Teknik Jaringan Komputer dan Telekomunikasi (TJKT), materi ini adalah tingkatan lanjut di mana kamu belajar bagaimana data dikelola, diamankan, dan didistribusikan.

Tanpa konfigurasi yang benar, kabel secanggih apa pun tidak akan berguna. Inilah dua pilar utama perangkat lunak (software) yang akan menjadi makanan sehari-hari siswa TJKT.

1. Sistem Operasi Jaringan: Berteman dengan Layar Hitam (Linux)

Lupakan tampilan Windows yang penuh warna. Di materi Administrasi Sistem Jaringan (ASJ), kamu akan berkenalan dengan Linux (biasanya distro Debian atau Ubuntu Server). Kebanyakan server di dunia maya tidak menggunakan mouse, melainkan baris perintah teks atau CLI (Command Line Interface).

Kamu akan belajar mengetik perintah "sakti" seperti:

root@server:~# apt update
root@server:~# nano /etc/network/interfaces
root@server:~# systemctl restart networking

Apa saja yang dibangun dengan Linux ini? Banyak sekali, di antaranya:

  • Web Server (Apache/Nginx): Membuat komputer kamu bisa menyimpan dan menampilkan website.
  • DNS Server (Bind9): Menerjemahkan alamat IP (misal 192.168.1.1) menjadi nama domain (misal sekolah.sch.id).
  • DHCP Server: Memberikan alamat IP secara otomatis kepada klien yang terhubung agar tidak perlu setting manual satu per satu.

2. Mikrotik: Router Sejuta Umat

Di Indonesia, hampir semua materi SMK TJKT menggunakan perangkat Mikrotik. Kenapa? Karena fiturnya lengkap, harganya terjangkau, dan sangat *powerful* untuk manajemen jaringan.

Fokus pembelajaran Mikrotik meliputi:

  • Manajemen Bandwidth (Queue): Pernahkah kamu merasa internet lemot karena ada satu orang yang sedang download file besar? Dengan Mikrotik, kamu belajar membatasi kecepatan setiap pengguna agar adil dan merata.
  • Hotspot & Voucher: Kamu akan belajar cara membuat sistem login Wi-Fi seperti di kafe-kafe, di mana pengguna harus memasukkan kode voucher untuk bisa internetan.
  • Routing (Static & Dynamic): Mengatur rute perjalanan data agar sampai ke tujuan dengan jalur tercepat.
Tahukah Kamu? Sertifikasi MTCNA (MikroTik Certified Network Associate) adalah salah satu sertifikat internasional yang paling diburu oleh siswa SMK sebelum lulus untuk meningkatkan nilai jual di dunia kerja.

3. Virtualisasi: Satu Komputer, Banyak Server

Di era modern, jarang sekali satu komputer fisik hanya diisi satu sistem operasi. Materi TJKT kini mencakup Virtualisasi menggunakan aplikasi seperti Proxmox atau VMware.

Teknologi ini memungkinkan kamu membuat banyak "komputer virtual" (VPS) di dalam satu komputer fisik. Misalnya, dalam satu PC server sekolah, kamu bisa menjalankan server ujian online, server website sekolah, dan server penyimpanan data sekaligus tanpa harus membeli 3 komputer berbeda.

Kesimpulan

Menguasai Administrasi Server dan Mikrotik membutuhkan logika yang kuat dan ketelitian tinggi. Salah ketik satu huruf saja dalam konfigurasi bisa membuat satu jaringan sekolah mati total (down).

Namun, keahlian ini sangat mahal harganya. Lulusan yang jago Linux dan Mikrotik biasanya menempati posisi System Administrator atau Network Engineer dengan gaji yang menjanjikan.
